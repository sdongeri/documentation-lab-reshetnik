# User Guide — Користувацька інструкція

## Початок роботи
1. Відкрий сайт у браузері.
2. Натисни кнопку “Реєстрація”.
3. Заповни форму та створи акаунт.
4. Увійди, використовуючи логін і пароль.

## Основні дії користувача
- перегляд списку фільмів на головній сторінці;
- відкриття сторінки фільму для читання оглядів;
- додавання власного огляду (після входу);
- редагування профілю.

## FAQ
1. Чи обов’язково реєструватися?
Ні, реєстрація потрібна лише для написання оглядів.

2. Чи можна редагувати свій огляд? 
Так, у своєму профілі.

3. Чи безкоштовний сервіс?  
Так, сервіс повністю безкоштовний.

4. Що робити, якщо я забув пароль?  
Скористайся функцією “Відновлення пароля”.
