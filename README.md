# documentation-lab-reshetnik
# Movie & Series Review Platform

Це вебсайт для перегляду та публікації оглядів фільмів і серіалів.
Проєкт допомагає користувачам швидко знайти якісний контент для перегляду,
орієнтуючись на відгуки інших людей.

## Проблема, яку вирішує система
- складно обрати фільм або серіал серед великої кількості варіантів;
- відгуки на різних платформах розкидані й неструктуровані.

## Основні функції
- перегляд списку фільмів і серіалів;
- читання оглядів та оцінок;
- реєстрація та вхід користувача;
- додавання власних оглядів;
- рейтинг популярних фільмів.

## Для кого цей продукт
- для глядачів, які шукають рекомендації;
- для кіноманів, які хочуть ділитися думкою;
- для студентських навчальних проєктів.
