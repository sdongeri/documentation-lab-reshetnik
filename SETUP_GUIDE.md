# Setup Guide — Встановлення та запуск

## Необхідні інструменти
- сучасний веббраузер (Chrome, Firefox, Edge);
- доступ до інтернету;
- GitHub-акаунт (для розробників).

## Запуск проєкту (макет)
1. Клонувати репозиторій:
   git clone https://github.com/username/documentation-lab-reshetnik
2. Відкрити папку проєкту.
3. Запустити файл `index.html` у браузері
   або перейти за посиланням на хостинг.

## Перевірка роботи
Проєкт запущено правильно, якщо:
- відкривається головна сторінка;
- відображається список фільмів;
- кнопки “Вхід” та “Реєстрація” працюють.
